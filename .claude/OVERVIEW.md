# AI Instructions - Project Overview Documentation

This file defines the structure for documenting a project overview. Follow this structure:

## Top-Level Sections
1. **Project Description** - Brief overview of what the project is
2. **Current Project State** - What's currently implemented and working
3. **Architecture** - Architecture explanation and package structure
4. **Design Patterns Used** - Patterns applied in the project
5. **Code Conventions** - Summary reference to this file
6. **Error Handling** - How errors are managed
7. **Dependencies** - External libraries and Java version
8. **Testing and Quality** - Testing status
